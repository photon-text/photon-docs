Build the PhotonText

Install prerequisites

To build PhotonText, you need to install the prerequisites described in the Introduction.

Build it

Have you installed all the prerequisites?

Once you have it installed, let's build it right away.

First, go to the directory where you downloaded the PhotonText source, then enter the following code in the terminal:

go build

Have you managed to build it? If so, try running PhotonText.

./photon
